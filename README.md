## 🚀 Features

### For Job Seekers
- **Smart Job Search** — Browse jobs with powerful filters (keyword, location, job type, experience level).
- **One-Click Apply** — Apply to jobs instantly with an optional cover letter.
- **Real-Time Application Tracking** — Track your application status through every stage (Pending → Reviewing → Shortlisted → Interview → Offered).
- **Save & Bookmark Jobs** — Bookmark interesting jobs for later.
- **Profile Management** — Manage your personal info, bio, phone, and resume URL.
- **Notifications** — Get notified when your application status changes.

### For Recruiters
- **Job Posting** — Create, edit, and manage job postings with detailed descriptions, requirements, salary ranges, and more.
- **Company Management** — Create and manage company profiles directly from the job posting form.
- **Applicant Tracking System (ATS)** — Review applications, view resumes and cover letters, and move candidates through hiring stages.
- **Dashboard Analytics** — Track total jobs, active jobs, total applications, and pending applications at a glance.

### For Admins
- **Platform Overview** — Monitor total users, recruiters, jobs, and applications across the entire platform.
- **User Management** — View all registered users and delete accounts when needed.

## 💻 Local Development

```bash
# Clone the repository
git clone https://github.com/Rajesh-Koppisetty/Nexus_Job_Board.git
cd Nexus_Job_Board

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open your browser and navigate to `http://localhost:5173`.
